Emoji Password Study (Coursework Prototype)
===========================================

Thank you for participating in this short usability study.

IMPORTANT:
- Do NOT use any real password that you use elsewhere.
- This is a coursework prototype.
- Data is stored locally on your computer.
- The study takes about 5â€“8 minutes.

----------------------------------------
How to Run (Windows)
----------------------------------------

1. Extract the zip file.
2. Double-click "run_emoji.bat".
3. A browser window should open automatically.
   If not, manually open:
   http://127.0.0.1:5000

4. Complete the study.
5. When finished, close the window.
6. Send the generated file "data.csv" back to the researcher.

----------------------------------------
Troubleshooting
----------------------------------------

If it says Python is not installed:
Download Python 3.10+ from:
https://www.python.org/downloads/

If the page does not load:
Make sure no other program is using port 5000.

----------------------------------------
What Data Is Collected?
----------------------------------------

- Time taken to create password
- Time taken to confirm password
- Login attempts and success
- Short questionnaire answers

No name or email is collected.
